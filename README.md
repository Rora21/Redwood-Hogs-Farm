# Redwood Hogs Farm

A Next.js website for Redwood Hogs Farm - Premium pig farming and quality livestock.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js 14 App Router pages
- `components/` - Reusable React components
- `public/images/` - Static images
- `styles/` - Additional styles

## Pages

- Home (`/`)
- About (`/about`)
- Pigs (`/pigs`)
- Services (`/services`)
- Gallery (`/gallery`)
- Contact (`/contact`)
